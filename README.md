# BA-CUSTOMER-BOOKING-PREDICTOR-MODEL
The trained machine learning model was deployed using Streamlit to create an interactive web application. Users can input booking-related features and receive real-time predictions on booking completion probability, enabling business teams to simulate customer behavior scenarios.
